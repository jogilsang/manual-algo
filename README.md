### 깊이/너비 우선탐색(DFS/BFS)
1. 그래프  
2. 방문했던 지점 다시 체크해야됨(안하면 무한루프)  
3. 너비우선탐색은 가까운곳부터 진행  
4. 깊이우선탐색은 막히지 않을떄까지 진행  
5. 깊이우선탐색은 모든걸 다 돌아야할수도있음  
6. 너비우선탐색은 모두 다 돌필요가 없음  
7. 깊이우선탐색보다는 너비우선탐색이 빠름    
8. 너비우선탐색은 최단거리 등 확인에 유효함
